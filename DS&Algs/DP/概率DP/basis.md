# 概率DP

## 核心思想

**概率DP的核心思想是将问题分解为子问题 并使用状态转移方程表示子问题之间的关系 同时考虑每个状态转移发生的概率。**

## 常见模型

- **概率计算模型: 计算某个事件发生的概率**
- **期望计算模型: 计算某个随机变量的期望值**
- **方差计算模型: 计算某个随机变量的方差**

## 相关公式

- **条件概率: P(A|B) = P(A ∩ B) / P(B)**
- **全概率公式: P(A) = ∑ P(A|Bi) * P(Bi)**
- **期望值: E[X] = ∑ xi * P(X = xi)**
- **方差: Var(X) = E[(X - E[X])²] = E[X²] - (E[X])²**
- **期望的线性性: E[aX + bY] = a * E[X] + b * E[Y]**
